# Projeto Java Blackjack / vinte-e-um

## Como rodar (versão principal)


### Dentro do VsCode, abra o terminal e acesse a pasta do projeto:

```
cd ProvaProjeto/
```


### Rode a aplicação:

```
java -cp bin ui.GameMain
```



## Como rodar (versão arquivo unico):

### Dentro do VsCode, abra o terminal e acesse a pasta do projeto:

```
cd ProvaProjeto/
```

### Compile o Codigo:

```
javac Blackjack.java
```

### Rode a aplicação:

```
java Blackjack
```
