## Como rodar (versão principal)


### Dentro do VsCode, abra o terminal e acesse a pasta do projeto:

```
cd ProvaProjeto/
```

### Crie a pasta onde ficarão os arquivos compilados:

```
mkdir bin
```

### Compile o Codigo:

```
javac -d bin $(find src -name "*.java")
```

### Rode a aplicação:

```
java -cp bin ui.GameMain
```



## Como rodar (versão antiga arquivo unico):

### Dentro do VsCode, abra o terminal e acesse a pasta do projeto:

```
cd ProvaProjeto/
```
